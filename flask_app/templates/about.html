About

Hello! This project aims to create a Pokemon fan club regarding the original 151 Kanto Pokemon.
The website features pages for each of the Pokemon which includes their basic info, abilities, and moves.
Users may create accounts to start commenting on their favorite Kanto Pokemon and use the favorite feature, which marks a single Pokemon as their favorite Pokemon which is displayed on their profile page.
